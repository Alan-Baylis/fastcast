About

Imagine if sharing video and audio content on the web had no boundaries and all that people needed were each other. Fastcast aims to make peer-to-peer file sharing easy to use and leave the rest for the community to make great content. The only limit to the amount of possibilities is the creativity of the individuals.

With BitTorrent, anything that doesn't already exist can be easily created by uploading the file to a web server that will take care of creating the torrent. WebRTC allows a streaming torrent client to run in a browser with pure JavaScript, so you won't have to install anything. People using torrent clients that support WebRTC will also be able to connect.
