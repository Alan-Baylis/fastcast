About

Imagine if sharing video and audio content on the web had no boundaries and all that people needed were each other. Fastcast aims to make peer-to-peer file sharing easy to use and leave the rest for the community to make great content. The only limit to the amount of possibilities is the creativity of the individuals. With BitTorent, anything that doesn't already exist can be easily created by uploading files to a web server which will take care of the web seeding. If the torrent swarm becomes too popular it would allow the web seed to be disabled. This is becuase people will be sharing the file while the browser page is open. WebRTC allows a streaming torrent client to run in a browser with pure JavaScript, so you won't have to install anything. People using torrent clients that support web seeds will also be able to transmit the file.
